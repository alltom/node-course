Summary

Congratulations! You're well on your way to creating some great web apps with Node.js. This mini-course has covered a lot:

  • Installing Node.js and third-party libraries
  • Making a web server with Express and extending it with middleware
  • Using socket.io for real-time client/server messaging
  • Using AJAX to fetch JSON generated on the server
  • Stashing data in redis and reading data from PostgreSQL
  • Authenticating usernames and passwords with Passport

It's kind of ridiculous. Still, there's a lot that hasn't been covered. Hopefully the links to the documentation on every page, and your newfound knowledge of the basics will give you some good legs to stand on.
