Make a Simple HTTP Server with Express

  1. Delete the console.log statement we have in app.js and replace it with the following code.

    Hover over the highlighted portions of the code to see what they do. 

    var express = require("express");
var app = express();

app.get('/', function (req, res) {
    res.end("Hello, world!");
});

app.listen(3000, function () {
    console.log('Server started at http://localhost:3000/');
});

    Node.js uses callback functions a lot. This code has two: one will be called when the server starts, and one will be called every time someone requests '/'.

  2. Run node app.js in the terminal again: 

    $ node app.js
Server started at http://localhost:3000/

  3. Does it work? Try visiting http://localhost:3000/. You should see "Hello, World!" on the page and nothing else.
